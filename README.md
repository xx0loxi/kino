# KinoFree - Movie Discovery App

Это приложение для поиска и просмотра информации о фильмах, созданное с использованием Next.js и The Movie Database (TMDB) API.

## Установка

1.  Установите зависимости:
    ```bash
    npm install
    ```

2.  Создайте файл `.env.local` в корне проекта (рядом с `package.json`) и добавьте ваш ключ API:
    ```
    NEXT_PUBLIC_TMDB_API_KEY=ваш_ключ_от_tmdb
    ```

## Где взять API ключ (Бесплатно)

1.  Зарегистрируйтесь на сайте [The Movie Database (TMDB)](https://www.themoviedb.org/signup).
2.  Перейдите в настройки аккаунта -> API.
3.  Создайте новый ключ (Developer).
4.  Скопируйте "API Key (v3 auth)".

## Запуск

Запустите сервер разработки:

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Ограничения

Этот проект использует легальное API для получения метаданных (постеры, описания). Для просмотра видео используется YouTube API (трейлеры). Прямой стриминг полных фильмов через бесплатные API обычно нарушает авторские права, поэтому в этом примере реализован просмотр трейлеров.
